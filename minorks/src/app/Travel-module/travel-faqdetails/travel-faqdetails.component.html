Travel Request FAQs



Q1.How do I raise Travel request?

All employees have access to raise travel request iff employee is allocated to travel onsite on particular project. Under "Initiate Travel request" menu user can raise a request by entering valid inputs.


Q2.Is a Travel Request required in order to travel?

Yes,one must raise a request to travel onsite since it has to be authorized by respective reporting manager and higher authorities.


Q3.Whom do I contact for assistance?

Once the raised request approved by your Manager, kindly contact Financial Head for further booking process.


Q4.Where do am I able to review the requests raised by me?

Employee is allowed to view the requests raised by them categorized by status(pending or approved or rejected) which is stored under "View Request" menu option.


Q5.Can I modify my travel request initiated?

Employee is allowed to modify his/her travel request which fall in the catagory of the status pending,approved by manager or confirmed by Finance.


Q6.Can I cancel my travel request initiated?

Employee is allowed to cancel his/her travel request which fall in the catagory of the status pending,approved by manager or confirmed by Finance.


Q7.Whether approval is required after modification/cancellation of requests?

Yes ,further approval is required if he/she Modifys any of the requests and in case of Cancel further approval is requried only when user cancels requests that falls in the category of status Confirmed by Finance.The notification of each action will be forwarded to respective approval authorities.


Q8.Can I re-call the request which has been rejected by higher authorities?

No, new request has to be raised.
